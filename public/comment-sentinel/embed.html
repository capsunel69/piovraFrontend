Import CSV

Upload comment data from CSV files

CSV Format

Your CSV should have columns for the comment data. Only comment_text is required, the rest are optional.

Download Template
comment_text *
The actual comment
author_name
Who wrote the comment
comment_date
When it was posted
likes
Reaction/like count
replies
Number of replies
post_url
Link to the original post
original_post
Text of the post the comment was on (used as context)

Column names don't need to match exactly -- you'll map them after upload. The system auto-detects common names like "text", "comment", "author", "date", etc.

Drop your CSV file here

Supports .csv, .tsv, .txt up to 50MB
