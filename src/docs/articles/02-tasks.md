## Tasks

Open **Tasks** from the side menu.

## What you can do

- Add **tasks** with a title, optional longer description, **priority**, and **due date**.
- Break work into **subtasks** (small checklist under a task).
- Use the **timer** on a task when you want to track focused time.
- **Mark done** or **re-open** a task without deleting it.
- **Delete** only when you want it gone for good.

## Working with the assistant

If you use chat, the assistant can add, update, or complete tasks **for you** when you ask clearly (e.g. “add a task…”).
